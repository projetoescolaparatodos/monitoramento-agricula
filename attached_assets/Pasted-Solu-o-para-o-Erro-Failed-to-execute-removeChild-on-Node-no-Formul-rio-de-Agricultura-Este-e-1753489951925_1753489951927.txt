Solução para o Erro "Failed to execute 'removeChild' on 'Node'" no Formulário de Agricultura
Este erro ocorre quando o React tenta manipular o DOM após um componente já ter sido desmontado. Vamos resolver isso de forma definitiva:

Análise do Problema
O erro acontece no componente FormAgricultura e está relacionado ao card.tsx. Especificamente, o React está tentando remover um nó do DOM que já não existe mais na árvore.

Soluções Implementar
1. Corrigir o Componente Card (card.tsx)
tsx
import React, { useEffect, useRef } from 'react';

interface CardProps {
  children: React.ReactNode;
  className?: string;
}

const Card = ({ children, className = '' }: CardProps) => {
  const cardRef = useRef<HTMLDivElement>(null);

  // Adicionar limpeza segura
  useEffect(() => {
    return () => {
      // Evitar manipulação direta do DOM
      if (cardRef.current && cardRef.current.parentNode) {
        // Nenhuma manipulação direta necessária
      }
    };
  }, []);

  return (
    <div 
      ref={cardRef}
      className={`bg-white rounded-lg shadow-md p-6 ${className}`}
    >
      {children}
    </div>
  );
};

export default Card;
2. Atualizar o Formulário de Agricultura (index.tsx)
tsx
import React, { useState, useEffect } from 'react';
import { useNavigate } from 'react-router-dom';
import Card from '../../components/ui/card';
import ErrorBoundary from '../../components/common/ErrorBoundary';

const FormAgricultura = () => {
  const [isMounted, setIsMounted] = useState(false);
  const navigate = useNavigate();

  // Garantir que o componente está montado
  useEffect(() => {
    setIsMounted(true);
    return () => setIsMounted(false);
  }, []);

  // Exemplo de função segura
  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    if (!isMounted) return;
    
    try {
      // Lógica de submit aqui
    } catch (error) {
      if (isMounted) {
        // Tratar erro
      }
    }
  };

  return (
    <ErrorBoundary fallback={<div>Erro no formulário</div>}>
      <Card>
        <form onSubmit={handleSubmit}>
          {/* Conteúdo do formulário */}
        </form>
      </Card>
    </ErrorBoundary>
  );
};

export default FormAgricultura;
3. Melhorar o ErrorBoundary
tsx
import React, { Component, ErrorInfo, ReactNode } from 'react';

interface ErrorBoundaryProps {
  children: ReactNode;
  fallback: ReactNode;
}

interface ErrorBoundaryState {
  hasError: boolean;
}

class ErrorBoundary extends Component<ErrorBoundaryProps, ErrorBoundaryState> {
  constructor(props: ErrorBoundaryProps) {
    super(props);
    this.state = { hasError: false };
  }

  static getDerivedStateFromError(_: Error): ErrorBoundaryState {
    return { hasError: true };
  }

  componentDidCatch(error: Error, errorInfo: ErrorInfo) {
    console.error('Erro no boundary:', error, errorInfo);
    // Adicione aqui o registro de erros (Sentry, LogRocket, etc.)
  }

  render() {
    if (this.state.hasError) {
      return this.props.fallback;
    }

    return this.props.children;
  }
}

export default ErrorBoundary;
Causas Comuns e Prevenção

tsx

  let isMounted = true;

    if (isMounted) setData(data);
  });

  return () => { isMounted = false; };
}, []);
